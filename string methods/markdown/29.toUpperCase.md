# The `toUpperCase()` method of String convert whole string into uppercase.


## Syntax
```
toUpperCase()

```

```javascript
const str = 'abcdefghijklmnopqrstuvwxyz';
console.log(str.toUpperCase());//ABCDEFGHIJKLMNOPQRSTUVWXYZ
```
