# The concat() method concatenates the string and returns the new string.
## Syntax
```concat(str1,str2,...n);
```
```Javascript 
const str1 = 'Cats are the best!';
const str2 = ' Yes OfCourse';
console.log(str1.concat(str2)); //Cats are the best! Yes OfCourse
```
