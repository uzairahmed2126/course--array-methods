# The padEnd() method adds a specified string to the end of the original string until it reaches a desired length.
## Syntax
```padEnd(targetLength)
padEnd(targetLength, padString)
```
```javascript
const str = '123';
console.log(str.padEnd(4,'45678'));//12345678
```
