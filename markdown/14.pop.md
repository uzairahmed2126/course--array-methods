# pop() 
## The pop() method remove the last element from the array and it's a mutable method.
```JavaScript
    const removeLast  = [1,2,3,4,5];
removeLast.pop;
console.log(removeLast);
```
### If you want to see deleted element then use it.
```JavaScript
const removeLast  = [1,2,3,4,5];
console.log(removeLast.pop());
```
